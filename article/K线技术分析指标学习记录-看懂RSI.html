2024-12-17 理财 复盘

RSI指标解释

RSI（相对强弱指数）是一个常用的技术分析指标，用来衡量一只股票在特定时间内涨跌的力度对比，可帮助我们判断股票是否超买或超卖。通俗点说，就是它是告诉你当前市场是不是涨太多了（需要回调） 或者 跌太多了（可能反弹） 。这个指标简单直观，短线交易非常常用，特别是震荡行情当中。

RSI的取值和计算方法

取值范围

RSI 的值在0 到 100之间：

高于 70：表示股票可能超买，即价格涨得太猛了，可能会回调。

低于 30：表示股票可能超卖，即价格跌得太狠了，可能会反弹。

中间区域（30~70）：一般没有明确的买卖信号，市场较为平稳。

RSI是怎么算的？

第一种传统计算公式

A 表示n天上涨幅度之后，B表示n天下跌幅度之和

RSI（n)=A/（A+B)×100%

第二种计算公式

  1. 计算一定周期内的涨幅和跌幅（以14天的周期为例）：

  2. RSI 计算最近 14 天的涨跌幅。

  3. 把这 14 天内每天的涨幅加总得到 平均涨幅。

  4. 同理，把 14 天内每天的跌幅加总得到 平均跌幅。

  5. 计算强弱比（RS）： $$ RS = \frac{\text{平均涨幅}}{\text{平均跌幅}} $$

比如，最近 14 天平均涨幅是 2，平均跌幅是 1，RS = 2 / 1 = 2。

  1. 计算 RSI： $$ RSI = 100 - \frac{100}{1 + RS} $$

如果 RS = 2 $$ RSI = 100 - \frac{100}{1 + 2} = 100 - 33.33 = 66.67 $$

简单的举个例子

假设某股票最近 14 天的收盘价涨跌如下(如果相比前一天是涨，就在涨幅这边加入，跌幅为0，同理，如果相比前一天是跌，在跌幅这边加入，涨幅为0)：

| 日期 | 涨幅 | 跌幅 | | ------ | ---- | ---- | | 第1天 | 2 | 0 | | 第2天 | 0 | 1 | | 第3天 | 3 | 0 | | 第4天 | 0 | 2 | | ... | ... | ... | | 第14天 | 1 | 0 |

$$ \begin{align} 平均涨幅 = \frac{2 + 3 + 1}{14} = 0.43 \ 平均跌幅 = \frac{1 + 2}{14} = 0.21 \ RS=\frac{0.43}{0.21} = 2.05 \ RSI=100 - \frac{100}{1 + 2.05} \approx 67.2 \end{align} $$

这意味着 RSI 处于 67 附近，接近 70，可能进入超买区

相比之下，第二种计算方式较为复杂， 但计算的RSI 的值更平滑，更加稳定

股票软件上的RSI指标解析

在没了解RSI指标之前，我是看不懂股票软件上的RSI指标的，比如日K线的RSI指标是RSI(6,12,24) RSI1:41.279 RSI2: 49.594 RSI3: 52.606这个是什么意思。在前面的学习案例里，我们使用的是14日为周期计算的。这里的RSI(6,12,24)其实就是多周期，代表了6，12，24三个不同的周期，后面的RSI1/RSI2/RSI3分别是对应周期的RSI值。
