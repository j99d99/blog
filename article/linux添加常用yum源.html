linux添加常用yum源

linux

添加epel源

#yum 直接安装
yum install -y epel-release

#收到安装
rpm -vih https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm

添加remi源

remi源里面几乎都是软件的最新稳定版

wget https://rpms.remirepo.net/enterprise/remi-release-7.rpm
rpm -Uvh remi-release-7.rpm
