加密模块hashlib

MD5加密和SHA1加密，方法类似 1. 就算一个字符串的md5值

import hashlib
import time

#加密字符串'123'和当前时间戳组成的字符串的md5值
ctime=time.time()
m = hashlib.md5(bytes('123'+str(ctime),encoding='utf-8'))

# 查看加密字符串
print m.hexdigest()
  1. 将一个字符串分段多次加密，结果也是一样 更上例一样
import hashlib
import time

#先计算字符串'123'的md5值
m = hashlib.md5(bytes('123',encoding='utf-8'))
# 在叠加计算当前时间戳的md5值
m.update(bytes(str(ctime),encoding='utf-8'))


# 查看加密字符串
print m.hexdigest()
